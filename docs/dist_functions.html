two decay functions: exponential decay and gaussian decay.

the decay functions look like under the distance range we have.

gaussian_decay = function(sigma, d){
  k = exp(-d^2/sigma^2/2)
  return(k)
}
  
exp_decay = function(sigma, d){
  k = exp(-d/sigma)
  return(k)
}
dist = seq(0, 2e7, by = 1e3)
